# Chapter 1: Creating the logic bricks!
## Sensor
Create Sensor Logic Brick: 
```bpy.ops.logic.sensor_add(type="",name='')```

Type:
* ACTUATOR
* ALWAYS
* COLLISION
* DELAY
* JOYSTICK
* KEYBOARD
* MESSAGE
* MOUSE
* MOVEMENT
* NEAR
* PROPERTY
* RADAR
* RANDOM
* RAY

## Controller
Create Controller Logic Brick:
```bpy.ops.logic.controller_add(type="",name='')```

Type:
* LOGIC_AND
* LOGIC_OR
* LOGIC_NAND
* LOGIC_NOR
* LOGIC_XOR
* LOGIC_XNOR
* EXPRESSION
* PYTHON
